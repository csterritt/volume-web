## Change the volume web application to be a mow.cli application

Use the mow.cli package to create a command line interface for the volume web application.
Documentation at https://github.com/jawher/mow.cli

The first command, 'serve', should start the web server.

The second command, 'weather', should display the current weather and forecast data,
hard-coded to my location (39.0438° N, -77.4874° W).
